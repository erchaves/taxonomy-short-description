Taxonomy Short Description

A WordPress plugin that shortens the term description shown in the administration panels for all categories, tags and custom taxonomies. This plugin does not effect public views of term descriptions at all.

Known Issues

  • You will not be able to sort by description in WordPress 3.1. #13650
